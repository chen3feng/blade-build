# 外部软件依赖

Blade 运行时需要以下依赖：

* Linux 或 Mac 操作系统
* Python v2.7+
* SCons v2.0+ or ninja v1.8+

Blade还能和以下软件协作：

* ccache v3.1+
* distcc

Blade 编译项目时可能需要到：

* gcc v4.0+
* jdk v1.6+
* scala v2.10+
* swig   v2.0+ (required for swig_library)
* flex v2.5+ (required for lex_yacc)
* bison v2.1+ (required for lex_yacc)
